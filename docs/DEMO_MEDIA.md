# Демо-материалы (видео и скриншоты)

План для **almost-full SaaS** презентации репозитория и продукта.

<a id="demo-script-4min"></a>

## Скрипт живого демо (~4 минуты)

Ориентир: **1080p**, без чувствительных данных. Порядок совпадает с **evaluator flow** в [README.md](../README.md#evaluator-five-minutes).

| Время | Шаг | Что говорить / показать |
|------|-----|-------------------------|
| 0:00 | **Контекст** | Коротко: multi-tenant copilot, RAG, workspace, async ingestion, квоты, аудит. |
| 0:20 | **Landing + pricing** | Главная гостя → прокрутка value prop → **/pricing** — планы Free / Pro / Team. |
| 0:45 | **Login / workspace** | Регистрация или вход → убедиться, что в шапке/сайдбаре выбрано **рабочее пространство**. |
| 1:00 | **Документы + upload** | **Документы** → **Загрузить** PDF/DOCX/TXT → закрыть тост, карточка со статусом. |
| 1:30 | **Async processing** | **Очередь обработки**: задача queued → processing → ready; вернуться в документы — **Готово**. |
| 2:00 | **Поиск** | **Поиск**: запрос по факту из файла → блок ответа и **источники** (чанки). |
| 2:30 | **Чат** | **Чат**: новый диалог, вопрос по содержимому → ответ с опорой на документы. |
| 3:00 | **Summary** | В **Документы** — иконка **Краткое содержание** на готовом файле → диалог summary. |
| 3:30 | **План и usage** | **План и лимиты**: тариф workspace, счётчики месяца; связка с [docs/quotas.md](quotas.md). |
| 3:50 | **Аудит** | **Аудит**: последние события (логин, загрузка, поиск — по факту среды). |
| 4:00 | **Закрытие** | Ссылка на GitHub, README, раздел «Быстрая оценка за 5 минут». |

Сокращённый вариант (**3 мин**): пропустить pricing или детальный разбор очереди; показать только переход «загрузил → готово → один запрос в поиске и в чате».

## Видео-демо (слот на сайте и в README)

1. Запишите экран по таблице выше.
2. Загрузите ролик на YouTube / Vimeo / корпоративный CDN.
3. **Локальный UI:** задайте переменную окружения **`NEXT_PUBLIC_DEMO_VIDEO_EMBED_URL`** — полный **URL для iframe** (например YouTube: `https://www.youtube.com/embed/VIDEO_ID`). Маркетинговая главная подставит плеер в секции «Видео-демо».
4. **README:** добавьте ссылку на тот же ролик (см. секцию «Демо-видео» в корневом README) — для тех, кто читает только репозиторий.

В README можно продублировать блок:

```html
<!-- Пример: замените src на свой embed URL -->
<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" title="Enterprise Copilot demo" allowfullscreen></iframe>
```

Требования к записи: **1080p**, **3–6 минут**, без утечек прод-данных (или с дисклеймером).
