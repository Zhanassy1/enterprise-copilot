# Демо-материалы (видео и скриншоты)

План для **almost-full SaaS** презентации репозитория и продукта.

## Видео-демо (слот)

1. Запишите экран: регистрация → выбор workspace → загрузка документа → ожидание статуса «Готово» → поиск или чат с источниками → «План и лимиты» → при необходимости «Аудит».
2. Загрузите ролик на YouTube / Vimeo / корпоративный CDN.
3. На маркетинговой главной (`frontend/src/components/landing/landing-page.tsx`) в секции **«Видео-демо»** замените заглушку на iframe или компонент с `src` вашего плеера.

Требования к записи: 1080p, 3–6 минут, без чувствительных данных (или с явным дисклеймером).
